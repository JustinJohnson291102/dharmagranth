# **App Name**: Dharmagranth TV

## Core Features:

- Live TV Streaming: Enable users to stream live Dharmagranth TV content across devices.
- Video Library: Offer a comprehensive video library categorized by genre, teacher, or topic.
- Program Guide: Provide a detailed program guide with schedules and descriptions.
- User Registration & Login: Allow users to create accounts and log in to access personalized content and features.
- AI-Powered Content Recommendations: The AI tool will analyze viewing habits and preferences to suggest relevant videos.
- Contact and Support: Provide a contact form and FAQs for user inquiries and support.

## Style Guidelines:

- Primary color: Warm saffron (#FF9933) to evoke spirituality and energy.
- Background color: Light beige (#F5F5DC), a soft and unobtrusive neutral.
- Accent color: Muted gold (#D4A27A) for highlighting key elements and creating a serene ambiance.
- Headline font: 'Literata', a transitional serif with a literary, vintage, slightly formal feel.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use simple, elegant icons representing spiritual concepts (e.g., lotus, OM symbol).
- Maintain a clean, grid-based layout with ample white space for ease of navigation and a sense of peace.
- Incorporate subtle transitions and loading animations to enhance the user experience.
